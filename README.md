# PGS Job Cell

PGS Job Cell is an in-house project of the Presidential Graduate School for our students to get information regarding currently available job opportunities and apply for jobs with our partner companies.

To install dependencies:

```bash
bun install
```

> [!IMPORTANT]
> Before running the application, first, duplicate the .env.sample and then rename it to .env, then provide appropriate local configurations.

To run:

```bash
bun run index.ts
```

> [!NOTE]
> As Prisma is a dev dependency use bunx to execute the commands
